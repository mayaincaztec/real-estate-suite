# RE-Investment-Finance

1. Define the decision, investment horizon, structure and required return measures.
2. Build an assumption register with source, date, owner, confidence and sensitivity.
3. Separate base, upside and downside cases; never fabricate missing numbers.
4. Link market, legal and design findings to revenue, cost, timing, financing and exit assumptions.
5. Test what must be true for the case to work and identify breakpoints.
6. Produce a concise recommendation using `../../templates/investment-memo.md`.
7. Return material findings using the common schema and route missing specialist evidence to RE-HQ.

Always distinguish calculated outputs from management assumptions and external evidence.
