# Transaction Document Family Map

Dùng reference này khi `legal-counsel` cần chọn đúng sub-mode theo family của tài liệu giao dịch.

## 1. Early-stage documents
Bao gồm:
- NDA
- LOI / Term Sheet
- exclusivity / process letter

### Tập trung vào
- binding vs non-binding;
- confidentiality;
- exclusivity;
- break-up mechanics;
- dispute / governing law.

## 2. Definitive deal documents
Bao gồm:
- SPA
- SHA
- JVA
- project transfer agreement

### Tập trung vào
- CP / closing mechanics;
- reps & warranties;
- indemnity / liability cap;
- termination / remedies;
- governance / control;
- project legal dependency nếu có.

## 3. Ancillary / support documents
Bao gồm:
- corporate approvals;
- powers of attorney;
- guarantees;
- side letters;
- disclosure schedules.

### Tập trung vào
- tính đủ và nhất quán với tài liệu chính;
- hidden risk do schedule / side document tạo ra;
- authority và enforceability.

## 4. Distressed / dispute-sensitive documents
Bao gồm:
- default notice;
- reservation of rights;
- settlement term sheet;
- waiver / amendment under stress.

### Tập trung vào
- trigger event;
- cure mechanics;
- prejudice / waiver risk;
- chứng cứ cần giữ.

## 5. Deliverable phù hợp theo family
- early-stage → clause issue list / recommendation memo
- definitive documents → contract review memo / legal question list / CP-closing issue note
- ancillary / support → legal question list / governance issue note
- distressed / dispute-sensitive → dispute note / recommendation memo
