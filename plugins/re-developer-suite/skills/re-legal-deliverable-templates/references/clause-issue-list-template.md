# Clause Issue List Template

Dùng khi cần output scan nhanh theo từng clause / issue / recommendation.

## Cấu trúc đề xuất

```md
## Clause Issue List
| Clause / Điều khoản | Nội dung chính | Vấn đề | Rủi ro cho client-side | Đề xuất sửa / fallback | Mức ưu tiên |
|---|---|---|---|---|---|
| ... | ... | ... | ... | ... | Must-Win / Negotiate / Accept |
```

## Khi không nên dùng
- khi cần reasoning đầy đủ dạng memo;
- khi vấn đề chính là thiếu facts / thiếu hồ sơ hơn là clause risk.
