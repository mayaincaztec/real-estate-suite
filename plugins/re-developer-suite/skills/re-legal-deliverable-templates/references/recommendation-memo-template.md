# Recommendation Memo Template

Dùng khi cần chốt hướng xử lý, so sánh option hoặc phát hành khuyến nghị hành động.

## Cấu trúc đề xuất

```md
## 1. Vấn đề cần quyết định
- ...

## 2. Kết luận khuyến nghị
- ...

## 3. Cơ sở chính của khuyến nghị
- ...

## 4. Các phương án khả dĩ
### Phương án 1
- Ưu điểm:
- Nhược điểm:
- Điều kiện tiên quyết:

### Phương án 2
- Ưu điểm:
- Nhược điểm:
- Điều kiện tiên quyết:

## 5. Rủi ro / giới hạn
- ...

## 6. Bước tiếp theo đề xuất
- ...
```

## Khi không nên dùng
- khi Sếp chỉ cần matrix hoặc checklist;
- khi facts còn quá thiếu để đưa khuyến nghị đáng tin cậy.
